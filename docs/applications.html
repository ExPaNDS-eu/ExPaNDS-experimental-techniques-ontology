Applications

PaNET provides persistent identifiers (PIDs) for experimental techniques in Photon and Neutron research. Thus it is an essential part of metadata in this field.

Applications

  • experimental data at large facilities: ongoing discussions with the NeXus community to define a standardized approch.
  • experimental data at lab devices: ongoing discussions with an industry partner to include PaNET in the lab raw data.
  • scientific publications: a BibTeX Citation Creator is available to create citations for individual terms.
  • beamline finder: proof of concept at DESY; a data base links the beamlines to the provided techniques and sample environment; a webpage allows access to the database and filter results
  • ICAT Metadata Catalog has a dedicated Technique table to hold PaNET terms. It has a many-to-many relation with Dataset, so you can tag datasets in ICAT with the experimental technique that has been used to create the dataset. There is also a contributed helper script to populate that Technique table from a PaNET OWL file.
  • PaNOSC search API can be hooked between the central search portal and the local metadata catalogues and will modify queries on the fly in such a way that conditions referring to a technique will be expanded to include all child techniques in PaNET.

Mappings

Several mappings from PaNET to other semantic artefacts (e.g., vocabularies) exist or are planned. A mapping connects terms from one dictionary to another and thus helps to reuse and extend existing use-cases and applications.

  • voc4cat (established)
  • Way for Light (ongoing)
  • IUCr dictionary (ongoing)
  • IUPAC gold book (ongoing)
  • Helmholtz Imaging (ongoing)
  • PhySH (ongoing)
  • wikidata (foreseen)
